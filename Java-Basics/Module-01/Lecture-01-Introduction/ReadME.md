# 🧠 Key Concepts

## print()

Prints output without moving the cursor to the next line.

Example:

```java
System.out.print("Hello");
System.out.print("World");
```

Output:

```
HelloWorld
```

---

## println()

Prints output and moves the cursor to the next line.

Example:

```java
System.out.println("Hello");
System.out.println("World");
```

Output:

```
Hello
World
```

---

## Compiler

Converts source code into machine-understandable code.

---

## Case Sensitive

Java treats uppercase and lowercase letters differently.

Example:

```java
age
Age
AGE
```

All are different.
